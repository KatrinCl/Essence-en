# 🌿 Essence SPA Website

A modern SPA salon showcase website with interactive service selection.

## 🔗 Demo

https://essence-rust.vercel.app

## ✨ Features

* 📌 Browse SPA programs and massages
* 👩‍⚕️ Choose masters with dynamic content updates
* 💾 Save selected services and masters (localStorage)
* 📱 Fully responsive design
* 📩 Contact form powered by Web3Forms
* 🎞️ Massage course slider

## 🛠️ Technologies

* React
* Tailwind CSS
* Vite
* Vercel (deployment)
* Web3Forms (form submission)

## ⚡ Implementation Highlights

* Uses `localStorage` to save selected services and masters
* Dynamic content update without page reload
* Mobile-friendly responsive layout
